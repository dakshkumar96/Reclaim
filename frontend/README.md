# Reclaim Frontend

Vintage dark-themed habit-building app built with React + Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the frontend directory:
```
VITE_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- Dark-only vintage aesthetic
- Notion-inspired clean design
- LeetCode-style structure
- Mobile-first responsive design
- Smooth animations
- JWT authentication
- Challenge tracking
- Leaderboard
- AI Coach chat

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router
- Axios
- Context API
